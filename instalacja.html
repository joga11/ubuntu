Instalacja maszyny wirtualnej VirtualBox z systemem Linux Ubuntu

1. Pobranie i instalacja VirtualBox

Windows:

1. Pobierz VirtualBox:

  • Przejdź na stronę pobierania VirtualBox: VirtualBox Download VirtualBox Download
  • Pobierz odpowiednią wersję dla systemu Windows.

2. Zainstaluj VirtualBox:

  • Otwórz pobrany plik instalacyjny i postępuj zgodnie z instrukcjami instalatora.

macOS:

1. Pobierz VirtualBox:

  • Przejdź na stronę pobierania VirtualBox: VirtualBox Download.
  • Pobierz odpowiednią wersję dla systemu macOS.

2. Zainstaluj VirtualBox:

  • Otwórz pobrany plik DMG i przeciągnij ikonę VirtualBox do folderu Aplikacje.

Linux (Ubuntu):

1. Dodaj repozytorium Oracle VirtualBox:

Otwórz terminal i dodaj repozytorium Oracle:

bash
Skopiuj kod
sudo sh -c 'echo "deb [arch=amd64]
https://download.virtualbox.org/virtualbox/debian $(lsb_release -cs) contrib" >> /etc/apt/sources.list.d/virtualbox.list'

2. Dodaj klucz publiczny Oracle:

bash
Skopiuj kod
wget -q https://www.virtualbox.org/download/oracle_vbox_2016.asc -O- | sudo apt-key add -
wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | sudo apt-key add -

3. Zaktualizuj listę pakietów i zainstaluj VirtualBox:

bash
Skopiuj kod
sudo sh -c 'echo "deb [arch=amd64]
sudo apt update
sudo apt install virtualbox-6.1

2. Pobranie obrazu ISO Ubuntu

1. Pobierz Ubuntu:

  • Przejdź na stronę pobierania Ubuntu: Ubuntu Download
  • Pobierz obraz ISO odpowiedniej wersji (np. Ubuntu 22.04 LTS).

3. Utworzenie Nowej Maszyny Wirtualnej

1. Uruchom VirtualBox:

  • Otwórz aplikację VirtualBox.

2. Utwórz nową maszynę wirtualną:

  • Kliknij przycisk "New" (Nowa).
  • Nadaj nazwę maszynie (np. "Ubuntu VM") i wybierz typ oraz wersję systemu operacyjnego (Linux, Ubuntu 64-bit).
  • Kliknij przycisk "Next" (Dalej).

3. Przydziel pamięć RAM:

  • Przypisz odpowiednią ilość pamięci RAM (zalecane minimum to 2048 MB, ale najlepiej 4096 MB, jeśli masz więcej pamięci).
  • Kliknij przycisk "Next" (Dalej).

4. Utwórz wirtualny dysk twardy:

  • Wybierz opcję "Create a virtual hard disk now" (Utwórz wirtualny dysk twardy teraz).
  • Kliknij "Create" (Utwórz).

5. Wybierz typ pliku dysku:

  • Wybierz VDI (VirtualBox Disk Image).
  • Kliknij "Next" (Dalej).

6. Dynamicznie przydzielany lub stały rozmiar:

  • Wybierz Dynamically allocated (Dynamicznie przydzielany).
  • Kliknij "Next" (Dalej).

7. Przydziel rozmiar dysku:

  • Ustal rozmiar wirtualnego dysku (zalecane minimum to 20 GB).
  • Kliknij "Create" (Utwórz).

4. Instalacja Ubuntu na Maszynie Wirtualnej

1. Skonfiguruj nośnik startowy:

  • Zaznacz nowo utworzoną maszynę wirtualną i kliknij "Settings" (Ustawienia).
  • Przejdź do sekcji "Storage" (Pamięć).
  • Wybierz kontroler IDE i kliknij ikonę dyskietki, aby dodać plik ISO.
  • Kliknij "Choose a disk file" (Wybierz plik dysku) i wskaż pobrany obraz ISO Ubuntu.

2. Uruchom maszynę wirtualną:

  • Kliknij "Start" (Uruchom).

3. Zainstaluj Ubuntu:

  • Po uruchomieniu maszyny wirtualnej z obrazu ISO, pojawi się instalator Ubuntu.
  • Wybierz "Install Ubuntu" (Zainstaluj Ubuntu).
  • Postępuj zgodnie z instrukcjami instalatora, aby skonfigurować system (wybierz język, układ klawiatury, ustawienia sieciowe, utwórz konto użytkownika itp.)

4. Zakończenie instalacji:

  • Po zakończeniu instalacji zrestartuj maszynę wirtualną.
  • Usuń plik ISO z nośnika startowego w ustawieniach maszyny wirtualnej.

5. Dodatkowe ustawienia i dostosowania

1. Zainstaluj dodatki gościa VirtualBox:

  • Uruchom maszynę wirtualną z Ubuntu.
  • W menu VirtualBox wybierz Devices > Insert Guest Additions CD Image... (Urządzenia > Wstaw obraz CD z dodatkami gościa).
  • Otwórz terminal w Ubuntu i zainstaluj dodatki gościa:
    bash
    Skopiuj kod
    sudo apt update
    sudo apt install build-essential dkms linux-headers-$(uname -r)
    sudo sh /media/username/VBox_GAs_*/VBoxLinuxAdditions.run
  • Zrestartuj maszynę wirtualną.

2. Dostosuj ustawienia maszyny wirtualnej:

  • Zwiększ pamięć wideo w ustawieniach maszyny wirtualnej.
  • Skonfiguruj wspólne foldery, jeśli chcesz wymieniać pliki między hostem a maszyną wirtualną.

To powinno wystarczyć, aby uruchomić i skonfigurować maszynę wirtualną VirtualBox z systemem Linux Ubuntu. Jeśli napotkasz problemy lub masz dodatkowe pytania, chętnie pomogę!
