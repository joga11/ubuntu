Wprowadzenie do Systemu Linux Ubuntu

Linux Ubuntu to popularna dystrybucja systemu operacyjnego Linux, znana z przyjaznego interfejsu użytkownika, stabilności oraz dużej społeczności użytkowników. Jest często wykorzystywana zarówno na komputerach osobistych, jak i na serwerach.

1. Podstawowe komendy Linux

1.1. Nawigacja w systemie plików:

  • pwd - Wyświetla bieżący katalog.
  • ls - Wyświetla zawartość katalogu.
  • cd - Zmienia katalog.
  • mkdir - Tworzy nowy katalog.
  • rmdir - Usuwa pusty katalog.

1.2. Operacje na plikach:

  • touch - Tworzy nowy, pusty plik.
  • cp - Kopiuje plik lub katalog.
  • mv - Przenosi lub zmienia nazwę pliku lub katalogu.
  • rm - Usuwa plik.
  • nano lub vi - Edytory tekstowe do edycji plików.

1.3. Informacje o systemie:

  • uname -a - Wyświetla informacje o systemie.
  • df -h - Wyświetla informacje o przestrzeni dyskowej.
  • free -h - Wyświetla informacje o pamięci.

1.4. Zarządzanie użytkownikami:

  • sudo adduser [nazwa_użytkownika] - Dodaje nowego użytkownika.
  • sudo passwd [nazwa_użytkownika] - Zmienia hasło użytkownika.
  • sudo deluser [nazwa_użytkownika] - Usuwa użytkownika.

1.5. Zarządzanie pakietami:

  • sudo apt update - Aktualizuje listę dostępnych pakietów.
  • sudo apt upgrade - Aktualizuje zainstalowane pakiety.
  • sudo apt install - Instaluje nowy pakiet.
  • sudo apt remove - Usuwa zainstalowany pakiet.

2. Ćwiczenia praktyczne

Ćwiczenie 1: Nawigacja w systemie plików:

  1. Otwórz terminal.
  2. Sprawdź bieżący katalog: pwd
  3. Przejdź do katalogu domowego: cd ~
  4. Wyświetl zawartość katalogu domowego: ls
  5. Utwórz nowy katalog o nazwie test: mkdir test
  6. Przejdź do katalogu: test: cd test
  7. Utwórz nowy plik o nazwie: plik.txt: touch plik.txt
  8. Wyświetl zawartość katalogu: test: ls
  9. Usuń plik plik.txt: rm plik.txt
  10. Usuń katalog test: cd .. && rmdir test

Ćwiczenie 2: Zarządzanie użytkownikami:

  1. Dodaj nowego użytkownika o nazwie student: sudo adduser student
  2. Zmień hasło użytkownika student: sudo passwd student
  3. Zaloguj się jako użytkownik student: su student
  4. Wyloguj się z użytkownika student: exit
  5. Usuń użytkownika student: sudo deluser student

Ćwiczenie 3: Zarządzanie pakietami:

  1. Zaktualizuj listę dostępnych pakietów: sudo apt update
  2. Zainstaluj edytor tekstu nano: sudo apt install nano
  3. Utwórz plik i edytuj go za pomocą nano: nano testfile.txt
  4. Zapisz zmiany i wyjdź z edytora (Ctrl+O i Ctrl+X).
  5. Usuń pakiet nano: sudo apt remove nano

Ćwiczenie 4: Informacje o systemie:

  1. Wyświetl informacje o systemie: uname -a
  2. Sprawdź użycie dysku: df -h
  3. Sprawdź użycie pamięci: free -h

3. Zarządzanie Usługami i Procesami

3.1. Systemd:

  • systemctl start [nazwa_uslugi] - Uruchamia usługę.
  • systemctl stop [nazwa_uslugi] - Zatrzymuje usługę.
  • systemctl restart [nazwa_uslugi] - Restartuje usługę.
  • systemctl status [nazwa_uslugi] - Wyświetla status usługi.
  • systemctl enable [nazwa_uslugi] - Umożliwia automatyczne uruchamianie usługi podczas startu systemu.
  • systemctl disable [nazwa_uslugi] - Wyłącza automatyczne uruchamianie usługi podczas startu systemu.

3.2. Zarządzanie procesami:

  • ps aux - Wyświetla listę wszystkich procesów.
  • top - Wyświetla dynamiczną listę procesów.
  • kill [PID] - Zakończenie procesu o podanym PID.
  • killall [nazwa_procesu] - Zakończenie wszystkich procesów o podanej nazwie.

Ćwiczenie 5: Zarządzanie usługami

  1. Sprawdź status usługi ssh: systemctl status ssh
  2. Uruchom usługę ssh: sudo systemctl start ssh
  3. Zrestartuj usługę ssh: sudo systemctl restart ssh
  4. Zatrzymaj usługę ssh: sudo systemctl stop ssh

Ćwiczenie 6: Zarządzanie procesami

  1. Wyświetl listę wszystkich procesów: ps aux
  2. Znajdź proces o nazwie bash: pgrep bash
  3. Wyświetl dynamiczną listę procesów: top
  4. Zakończ proces top naciskając q

Te podstawowe informacje i ćwiczenia praktyczne pomogą Ci lepiej zrozumieć i administrować systemem Linux Ubuntu. W miarę jak będziesz zdobywać doświadczenie, możesz zgłębiać bardziej zaawansowane tematy, takie jak zarządzanie sieciami, bezpieczeństwo systemu, skrypty shellowe i wiele innych.
